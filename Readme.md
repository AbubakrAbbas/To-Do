# ToDo — Kanban Task Manager

A fully functional browser-based Kanban-style task manager built with pure vanilla JavaScript, HTML5, and CSS3. No frameworks, no libraries — just clean client-side code.

![ToDo Kanban App](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

🔗 **Live Demo:** [abubakrabbas.github.io/To-Do](https://abubakrabbas.github.io/To-Do/)

---

## Features

- **Three Kanban columns** — Todo, In Progress, and Completed
- **Add tasks** with a title and description via a clean modal
- **Drag and drop** tasks between columns
- **Delete tasks** with a single click
- **Live task count** updates per column automatically
- **localStorage persistence** — tasks are saved and restored on page refresh
- **Responsive design** — works on desktop, tablet, and mobile
- **Bottom sheet modal** on mobile for a native app-like feel

## Getting Started

No installation or build steps needed.

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/todo-kanban.git
   ```

2. Open the project folder:

   ```bash
   cd todo-kanban
   ```

3. Open `index.html` in your browser — that's it.

---

## Usage

1. Click **+ Add new Task** in the top right
2. Enter a title and description
3. Click **Add Task** — it appears in the **Todo** column
4. **Drag** the task card to **In Progress** or **Completed** when ready
5. Click **Delete** on any card to remove it
